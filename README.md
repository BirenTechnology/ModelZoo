# 欢迎使用 BIREN ModelZoo
本仓库是由壁仞科技(BIRENTECH)开发和维护的模型合集，提供基于壁仞硬件产品以及壁仞软件栈(BIRENSUPA)的模型训练和推理代码示例。

通过参考本仓库中的源码，开发者可以了解壁仞软件栈(BIRENSUPA)的编程最佳实践，获取基于壁仞软硬件产品的模型移植和性能调优参考。
